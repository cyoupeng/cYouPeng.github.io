Submit New Application

To modify or view your application, click on the file number. Please note: Depending on the stream you choose, you will need to complete between nineteen to twenty-two sections before you can submit. The application will take a number of hours to complete. If you wish to withdraw your application, click on Withdraw. Please note that a withdrawn application is non-refundable if the OINP has already started processing it.

If you see the red alert icon beside your file number, the OINP requires additional documentation. Click on the icon for more information about the request.

When all document requests are submitted successfully, a blue document icon will appear beside your file number. Clicking on the blue document icon will show details of all submitted document requests.

If you see the green checkmark beside your file number, the OINP does not require any additional information or documentation from you at this time.
